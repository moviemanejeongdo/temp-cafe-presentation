냥냥 카페 1호점

팀명: 알바생들

가상현실(VR)에서 AI NPC와 소통하는 수치사 유발 시뮬레이션!

🐾 STT 성능 검증실 열기 🎤

✨ 프로젝트 컨셉 및 핵심 기믹

  • 컨셉: 테이크아웃 전문 고양이 메이드카페 (VR 환경)
  • 핵심 1: 사용자가 마이크로 "냥냥카페다 냥!" 외쳐야 게임 진행
  • 핵심 2: Gemma4 LLM을 이용한 STT/TTS NPC 상호작용
  • 핵심 3: VR 조이스틱을 활용한 현실적인 커피 음료 제조 HUD

👥 역할 분담 세부 목표

  • 조예은: UI, Git 브랜치 구축, 우유/물, 시럽 시스템
    문서작업(와이어프레임)
  • 최재빈: NPC LLM 대사, 점수에 맞는 LLM 평가, 그라인더, TTS, STT 시스템 연동
    문서작업(콘티)
  • 황인용: 플레이어 상호작용, 이동, 샷, 탬핑, 스팀, 오브젝트 이펙트
    문서작업(워크플로우)
  • 정해륜: 맵 디자인, 조이스틱 데이터 모션 인식, 점수에 따른 npc 모션, 얼음 시스템, 손님에게 음료 건네기 액션
    문서작업(워크플로우)
  • 다같이: 카페 전체 맵 디자인 및 3D 에셋(Asset) 서칭 배치

🎮 VR 인게임 플레이 시나리오

  • 1. 주문받기: 홀로그램 UI를 통해 LLM NPC와 대화
  • 2. 음료제조: 1인칭(FPS) 시점으로 VR 조이스틱 조작
  • 3. 서빙하기: 완성된 음료를 손님 NPC에게 건네기
  • 4. 평가받기: 제조 스탯을 채점하여 손님의 칭찬!
